# CSS

### overflow 속성

> 요소의 박스에 내용이 더 길 때 어떻게 보일지 선택하는 속성
>
> 상속되지 않음

- visible: 기본 값으로, 내용이 더 길어도 그대로 보임 (흘러넘침)
- hidden: 내용이 넘치면 자름. 넘치는 부분은 보이지 않음
- scroll: 내용이 넘치지 않아도 항상 스크롤바가 보임
- auto: 내용이 잘릴 때만 스크롤바가 보임
